  • 专业介绍

广东省特色专业：电子信息工程专业的特色与优势 电子信息工程专业源于1979年第一届招生的应用电子技术专业，国家专业代号080701，广东省特色专业，广东海洋大学第一个教育部专业认证受理专业（2020），国家一流专业培育专业，负责农业工程与信息技术专业硕士学位点建设。 本专业师资力量雄厚，其中高级职称10人，外籍博士后1人，具有博士学位教师15人。
